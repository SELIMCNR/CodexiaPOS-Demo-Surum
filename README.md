# CodexiaPOS (Demo Sürüm)

CodexiaPOS, küçük işletmeler için modüler, kullanıcı dostu ve güvenli bir satış otomasyon sistemidir. Bu demo sürüm, temel özellikleri içerir:

## Özellikler
- Ürün ekleme ve satış
- Kasa işlemleri
- Rapor ekranı
- Tema ve dil desteği
- Giriş ekranı ve kullanıcı doğrulama

## Kurulum
1. Python 3.10+ kurulu olmalı
2. Gerekli paketleri yükleyin: pip install PyQt6
3. `main.py` dosyasını çalıştırın

## Pro Sürüm
Pro sürümde aşağıdaki özellikler yer alacaktır:
- Gelişmiş kasa ve raporlama
- Kullanıcı yönetimi
- İndirim, iade, kampanya
- WhatsApp ve Instagram entegrasyonu
- Yapay zeka destekli satış tahmini

## Lisans
Bu demo sürüm açık kaynaklıdır. Pro sürüm ticari lisansla sunulacaktır.
